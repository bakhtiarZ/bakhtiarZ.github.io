ProPutter is an embedded golf-training system that captures swing data from a club-mounted sensor module and turns it into actionable feedback. The project combined motion sensing hardware, signal interpretation, and a web interface for reviewing performance over time.

The platform was designed to measure features such as swing speed, path, and club orientation, then surface those signals through analytics and coaching-oriented feedback. Beyond the device itself, the project also included a companion web application for presenting results and supporting a broader user workflow.

This was a multidisciplinary build that sat at the intersection of embedded systems, data analysis, and user-facing software.
